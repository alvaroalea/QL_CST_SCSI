# CST SCSI/FLOPPY CONTROLLER for Sinclar QL

A clone of the CST's Floppy + hardisc Interface for the Sinclair QL

# WARNING: DO NOT USE THIS PROJECT IS NOT TESTED AND MISSING THINGS

An evolution of the CST V4 Floppy interface with the incorporation of SCSI Controller.

The SCSI was limited only to a especific HDD model: Rodine RO652 20Mb, an UK Manufactured HDD that was also used on Macintosh SE

Later this interface will be use as a base for the Thor 20 Computer SCSI interface.
